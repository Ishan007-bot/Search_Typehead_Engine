# Screenshots

Drop the images referenced by the main [README](../../README.md) here. Use these
exact filenames so the links resolve:

| File | What to capture |
|---|---|
| `suggestions.png` | The UI with a prefix typed (e.g. `weath`) showing the ranked dropdown + popularity bars. |
| `trending.png`    | The "Trending now" chips after running a few searches. |
| `cache-debug.png` | Browser/terminal showing `p` — owner node, hit/miss, sample distribution. |
| `batch-stats.png` | `GET /batch/stats` after a burst of searches — `searches_enqueued` vs `db_flushes` (the write reduction). |

PNG or JPG both render on GitHub. Keep them reasonably sized (< ~500 KB each).
